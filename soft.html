セキュリティソフトの必要性と未来大生の現状

Windows10に標準搭載されている「Windows Defender」。無料で搭載されているものだが性能は悪くなく、必要な機能のおおよそは揃っている。
具体例としては、ウイルスやマルウェア（悪意のあるソフトウェア）などの脅威を継続的にスキャンし、保護してくれる。[1]
他にも様々な機能が搭載されているので「有料のセキュリティソフトをわざわざ購入してインストールする必要が無い」と思う人もいるだろう。

では、Windows Defenderに加えて、有料のソフトを利用する意味や利点はなんだろうか。
それは「セキュリティレベルの強化」と「便利な機能の追加」である。

Windows Defenderの

  • x未知のマルウェアを防ぐことが出来ない危険性がある
  • x迷惑メール防止やネットバンキング保護ができない
  • xID/パスワード管理機能や保護者見守り機能などの便利機能が存在しない

というような課題を有料のソフトを使用することで解決できる。
Windows Defenderのみに頼らず、良質な有料ソフトを導入した方がより安全、より快適なものになるだろう。


実際にメンバーが体験

初期搭載されているWindows Defenderと、市販されている様々なセキュリティソフトにどのような違いがあるのか。代表的なセキュリティソフト５種を、本プロジェクトのメンバーが実際に使用したレビュー付きで紹介します。未来大生ならではの感想も含まれているので、是非ご一読ください。

Windows Defender

特徴
  • ◎Windowsに初期から搭載されている[1]
  • ◎ウイルスに感染していないかどうか探すスキャン保護[1]
  • ◎ランサムウェアからの保護強化機能を搭載[2]
  • ◎不正なWebサイトをブロックする"SmartScreen"[3]
  • ◎脆弱性を保護する"Exploit Protection"[4]

注意点
    x誤検知の可能性がある[5]
    x迷惑メールや不適切な広告を防ぐ機能はない
    x未知のウイルスには対応できない可能性がある

ESET インターネット セキュリティ

特徴
  • ◎無料期間：最大2ヵ月[6]
  • ◎「ペアレンタルコントロール」機能でWebフィルタリング、アプリの利用制限をかけられる[7]
  • ◎軽さに定評があり、「友人や同僚に薦めたい」セキュリティソフトNo.1を3年連続で受賞[7]
  • ◎Googleにも認められた検出技術[7]
注意点
  • x毎月1日に課金が始まる仕組みのため、開始日によっては無料期間が短縮される[6]
  • xMacは一部の機能が非対応[7]
  • xパスワード管理とデータ暗号化の機能は、上位版（プレミアム）の製品にのみ搭載[8]


阿保信宏（情報システムコース３年）のレビュー
【使用環境】　　Windows10 HOME(推奨機), Intel(R) Core(TM) i3-7100U CPU, 8.00GB
【ソフト情報】　有効期限１年, 利用可能台数１台, ダウンロード版, 6000円
【使用頻度】　　毎日
【主な利用目的】課題や動画サイトの閲覧

Q.良いところは何ですか？
主張が強すぎないので、普段使いに影響が少なかった。また、メールもアップデートなど重要なものしか送られてこないので煩わしさが無かった。
Q.悪いところは何ですか？
他のセキュリティソフトのアンインストールを求められるのだが、Windows Defenderは含まれないことの説明が無かったので混乱する人もいそうだと思った（Windows用の案内ページなので尚更）。
Q.性能的にこの値段が妥当だと思いますか？
妥当だと思う。難しい事や面倒な事があまり無いので、あまり知識が無くても使いやすい。
Q.未来大生が使用することはオススメできますか？
オススメできる。Windows Defenderだけでは不安だという人は使ってみてはどうだろうか。

マカフィー リブセーフ

特徴
  • ◎無料期間：30日間[9]
  • ◎インストール台数無制限[9]
  • ◎安心のウイルス検知率100％[10]
  • ◎常に最新な状態に自動で更新してくれる[10]
  • ◎ウイルス対策だけでなく個人情報を守る様々な機能も搭載[10]
  • ◎ストレージクリーナー、メモリーブースター等のモバイル端末最適化の機能も搭載[11]
注意点
  • xマカフィー製品は様々なサイトで販売されており、有効期限を延長する際は同じ購入元かつ、同じ製品を買わなければ延長されない[12]
  • xカメラ等を使用するアプリ（例：Zoom, Skype）を使う場合、ファイアウォールによってブロックされていると正しく機能しないことがあるので、その場合はプログラムの権限リストに登録する必要がある[13]
髙橋大輔（複雑系コース３年）のレビュー
【使用環境】　　Windows10 HOME(推奨機), Intel(R) Core(TM) i3-7100U CPU, 8.00GB
【ソフト情報】　有効期限１年, 利用可能台数１台, ダウンロード版, 5980円
【使用頻度】　　毎日
【主な利用目的】課題や講義、動画サイトの閲覧

Q.良いところは何ですか？
ウイルス対策やセキュリティの面での性能は高く感じた。特にウェブの安全性を評価ツールにより、検索画面で検索結果を自動で評価しファイアウォールやウイルス対策が正常に動いているかを確認したり、ウイルス対策として、ダウンロードした際に危険がないかをも自動でチェックしてくれる。
Q.悪いところは何ですか？
ウイルス対策として自動でウイルススキャンが行われるが、授業の課題でコマンドプロンプトを使ってC言語のコンパイルを行ったとき、特にファイルの読み書きをしたところ、実行したC言語ファイルがウイルススキャンに引っ掛かり実行できないという状態になった。また、検索エンジンが自動的にYahooに設定されるので、Chromeなど他の検索エンジンを使う場合は自分で設定しなおさなければならない。
Q.性能的にこの値段が妥当だと思いますか？
自分の環境での使用では値段に見合った性能とは思わなかった。
Q.未来大生が使用することはオススメできますか？
学生が学校生活で使うのには不向きだと感じた。ゲームPCなどの個人PCにいれた方が適していると感じた。

カスペルスキー セキュリティ

特徴
  • ◎無料期間：30日間[14]
  • ◎防御力や使いやすさの総合満足度が95.8%（2020年11月20日時点）[15]
  • ◎検知率100%、誤検知0件[15]
  • ◎多彩な防御機能[15]
    ・危険なサイトや広告、ランサムウェア、迷惑メール、フィッシング詐欺、不正送金等の被害から守る
    ・子供の利用時間制限や悪質サイトへのアクセスブロック（ペアレンタルコントロール）
  • ◎有効期限中はPCやモバイルを買い換えても継続利用できる[15]
  • ◎悪意のあるソフトウェア（ランサムウェア含む）の動作を追跡し、システムの改ざん等がされた場合に変更をロールバックする"システムウォッチャー"[16]
注意点
  • xID/パスワード、クレジットカード情報、大切なメモや画像などのデータを暗号化して保管できるパスワード管理ツールがMacに非対応[17]
深谷哲平（知能システムコース３年）のレビュー
【使用環境】　　Windows10, AMD Ryzen 7 3700X 8-Core Processor, 16GB
【ソフト情報】　有効期限１年, 利用可能台数１台, パッケージ版, 4054円
【使用頻度】　　毎日

最初のスキャン以外特に自分で操作する点がない。また、動作に支障がなく、広告をブロックしてくれる。ソフトの専用アプリが使いやすい。
Q.悪いところは何ですか？
追加で料金が発生するものを推奨してくる。
Q.性能的にこの値段が妥当だと思いますか？
他の製品と比べて値段も高くないので妥当だと思う。
Q.未来大生が使用することはオススメできますか？
推奨機で使用したわけではないので全員に当てはまるかはわからないが、課題をやっている中で不便もなく動作に支障もないのでおすすめしてもいいと思う。

ウイルスバスタークラウド

特徴
  • ◎無料期間：30日間[18]
  • ◎無料期間終了後、自動的に有償版にならない親切仕様[18]
  • ◎1000万人以上の利用者数[18]
  • ◎独自の情報収集＋外部の専門機関との連携により、ネット詐欺対策の検出力を強化[18]
  • ◎未知のWebサイトに利用者がアクセスした際、詐欺サイトかどうかAIが検出して判断してくれる[18]
  • ◎TwitterやFacebookなどのSNSのリンク先の安全性を評価[19]
  • ◎"XGen"（クロスジェネレーション）セキュリティは防御力の最大化（防御のスマート化）を目指すとともに、最適化されたセキュリティを提供することでIT環境への影響の最小化したり、システム全体で連携し防御・検知・対応の迅速化を目指している。[20]

注意点
  • x申し込み当月の解約、および解約当月の再申し込みは不可[21]
  • xクレジットカード決済のみ対応[21]
中村一貴（複雑系コース３年）のレビュー
【使用環境】　　macOS Mojave, Intel(R) Core(TM) i7 CPU, 16GB
【ソフト情報】　有効期限１年, 利用可能台数３台, パッケージ版, 5200円
【使用頻度】　　毎日
【主な利用目的】講義（課題）から趣味など日常的に利用する

Q.良いところは何ですか？
ブロックした際の通知やその他の警告が少なく、快適に利用することができた。またほとんどの処理をクラウド上で行なっているため、動作が重くなることもなかった。
Q.悪いところは何ですか？
実際にどのファイルをどれくらい見てくれているのかがわからない。加えて、クラウド上の処理であることから回線の弱い場所では、動作が極端に重くなってしまう。
Q.性能的にこの値段が妥当だと思いますか？
妥当である。その理由として、使ってみてなにも問題がなかったことと、動作の快適さがよかった。
Q.未来大生が使用することはオススメできますか？
1年生で初めて自分のPCを買った人にはおすすめできる。なにもわからず、とりあえずなにか入れておきたいという人なら不自由なく利用できると思う。

ノートン360

特徴
  • ◎世界売上シェアNo.1で、60日間返金保証[22]
  • ◎ドイツの第三者評価機関であるAV-TESTが実施したセキュリティソフトの保護力テストで、4年連続最優秀賞を受賞[22]
  • ◎「アンチウイルス」「ゼロデイ脆弱性保護」「ファイアウォール」など、マルウェアから守るための機能が豊富にある[23]
  • ◎既知のマルウェアだけでなく、未知のマルウェアからパソコンを守る「ふるまい検知(SONAR)」も搭載されている[23]
  • ◎小さい子供がいる親向けのノートンファミリーは、様々な機能によって教育の手助けや不適切なサイト等から子供を守ってくれる[24]
注意点
  • x「クラウドバックアップ機能」や「セーフカム機能」はWindowsのみ[25]
  • xモバイル版はAndroid6.0以上、iOSは最新および2バージョン前のみの対応で、古い機種は使用不可[26][27]
